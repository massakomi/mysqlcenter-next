# Фронт для MySQLCenter - программы для управления базой данных MySQL

## Запуск проекта в режиме разработки

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Запуск проверки статическим анализатором

```bash
next lint
```

## Запуск форматирования кода на всех файлах

```bash
npx prettier . --write
```
